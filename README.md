# 📚 Library Event Producer API

Spring Boot microservice producing Kafka events for book registrations in a library system. Java 21 + Gradle with Dockerized services & full test suite. 🚀

## 🛠️ Tech Stack
- **Backend**: Spring Boot 3.x, Java 21 ⚙️
- **Messaging**: Kafka Producer/Consumer APIs 📨
- **Build**: Gradle 🔨
- **Containerization**: Docker + docker-compose 🐳
- **Database**: PostgreSQL (consumer side) 🗄️
- **Testing**: JUnit 5, Testcontainers, Integration Tests ✅
